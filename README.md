# 🕵️‍♂️ Ứng dụng Phát hiện Giao dịch Gian lận (Fraud Detection App)

Đây là ứng dụng web tương tác được xây dựng bằng **Streamlit**, chuyển đổi từ Notebook Machine Learning chuyên phát hiện các giao dịch tài chính gian lận dựa trên tập dữ liệu chuẩn.

Ứng dụng hỗ trợ các thuật toán như **Random Forest** và **Logistic Regression** với khả năng tiền xử lý dữ liệu động, mã hóa (Label Encoding), và chuẩn hóa (StandardScaler) được lưu trữ liên kết suốt phiên hoạt động.
